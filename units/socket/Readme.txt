Telnet daemon and telnet client for KOL with mirrors.
By Alexander Shakhaylo, alex_s@kcci.kharkov.ua

INSTALLATION
1. Unpack where You wish preserving ralative pathes.
2. Create package (e.g. KOLSOCKPAK.DPK), add mckSocket.pas there.
3. Change package options to "Design-time only" and "Rebuild as needed".
4. Save it, compile and install it.
5. Compile projects. Do not forget to add KOL_MCK conditional define
   in project options.

Enjoy!
