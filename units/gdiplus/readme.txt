***************
KOL GDI OBJECTS
***************

Very simple but powerfull GDIPLUS.DLL encapsulation

GDIPLUS is the replacement/enhancement of the oldstyle GDI.DLL
and contains new Pens, brushes, fonts en imagehandling for JPG,GIF,PNG,EMF,WMF and much, much more.

Written by Thaddy de Koning for KOL based on code by
Dave Jewell for PCPRO and The Delohi Magazine.

Freeware

The unit KOLGDIV2.pas contains all objects
The rest of the files are for the demo.
