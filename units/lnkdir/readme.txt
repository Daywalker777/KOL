Предназначен для определения расположения системных папок, таких как "Мои документы",
"Автозагрузка", "Рабочий стол", "Меню пуск" и др.

Функции:
 GetProgramsDir - Program files
 GetStartUpDir - Автозагрузка
 GetStartMenuDir - Меню ПУСК
 GetDesktopDir - Рабочий стол
 GetFavoritesDir -  Избранное
 GetFontsDir - Шрифты
 GetHistoryDir - История
 GetMyDocumentDir - Мои документы
 GetSendToDir - SendTo
