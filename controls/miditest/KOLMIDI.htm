Thaddy’s KOL Midi example

This demo is written using the KOLMIDIOUT object by Thaddy de Koning. It features a full midi implementation. It is based on the JCL midi library by Robert Rossmair, but completely rewritten for use with KOL. Instructions

The first slider is Pitch control.

The second slider is Modulation

Midi out selects the midi device

Program selects the instrument

Octave selects the highest full octave available

Notes off is a panic button

The blue field indicates the current note played

Control is with the mouse.
