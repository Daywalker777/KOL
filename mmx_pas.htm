type TCpuId =( cpuOld486, cpuNew486, cpuMMX, cpuSSE, cpuSSE2 );
Enumeration type to represent CPU type. cpuOld486: Old 486 Processor and earlier cpuNew486: New 486 Processor to Pentium1 without MMX cpuMMX : MMX supported (but not SSE or SSE2) cpuSSE : MMX and SSE supported (but not SSE2) cpuSSE2 : MMX, SSE and SSE2 supported

function GetCPUType: TCpuId;
Checks CPU (Intel PC x86 Architecture) for MMX support.
