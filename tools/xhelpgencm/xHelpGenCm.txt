Compact KOL Help Generator v3.0

1. Запускает браузер по умолчанию.
2. Создает заново документацию, если вы удалили все HTM-файлы.
3. Элементы левого фрейма сортируются по алфавиту.
4. По умолчанию не создается справка для предка объекта, а
   только ссылка на него.

Чтобы создавалась справка для предка объекта в XHelpGen.ini:
[Settings]
ParentHelp=1 (по умолчанию 0)
